Panduan & Alur Kerja

Panduan Identitas (User ID)

Tidak Ada Registrasi Mandiri

Aplikasi ini tidak menyediakan pendaftaran akun mandiri. Setiap pengguna wajib menggunakan User ID yang diperoleh setelah registrasi di aplikasi SmartBank (Kelompok 1).

Alur Kerja Transaksi
1

Daftar & Isi Saldo di SmartBank

Lakukan pendaftaran dan pengisian saldo di sistem SmartBank terlebih dahulu.

2

Pesan via Platform Terhubung

Pesan layanan LogistiKita melalui platform terhubung seperti Marketplace atau langsung di website ini.

3

Pembayaran Otomatis via Integrator

Setiap transaksi pembayaran diproses secara otomatis melalui Integrator (API Gateway) untuk memotong saldo SmartBank Anda.

4

Lacak Paket Anda

Gunakan User ID yang sama untuk melacak status paket melalui fitur Tracking di halaman ini atau di dashboard.

Dukungan Sistem: Integrasi pembayaran dan validasi keamanan sepenuhnya dikelola secara terpusat demi menjamin keamanan saldo Anda.
